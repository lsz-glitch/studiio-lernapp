# Backend wurde verschoben

Der API-Server liegt jetzt unter **`backend/`**.

- Start: `npm run api` (startet `backend/index.mjs`)
- Konfiguration: `backend/.env` (siehe `backend/.env.example`)
- Falls du hier eine `.env` hattest, kopiere den Inhalt nach `backend/.env`.

Siehe **`backend/README.md`** und die Haupt-**`README.md`** im Projektroot.
